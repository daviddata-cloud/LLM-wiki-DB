# Digital Information Asset Classification Framework
The Centers for Disease Control and Prevention (CDC) established the official framework for classifying all digital information assets. This framework aims to ensure the secure handling of sensitive data.
[[public-health-emergency-data-handling]] provides guidance on pandemic response data handling, but this document focuses specifically on CDC's internal classification procedures.

# CDC EARS classification procedures are mentioned in this framework. For more information on EARS, see [[ears]].
